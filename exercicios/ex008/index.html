Principais formatações

Negrito e destaque

Para inserir um texto em negrito usamos a tag b. (não semântica)

Para inserir um texto em destaque utilizamoa a tag strong. (semântica)

Itálico e ênfase

Para inserir um texto em itálico utilizamos a tag i. (não semântica)

Para inserir um texto em ênfase utilizamos a tag em. (semântica)

Texto marcado

Para inserirmos um texto marcado usamos a tag mark. Ela vem na cor amarela
padrão mas é possível mudar usando a css.

Texto grande e pequeno

Para inserirmos um texto grande utilizamos a tag big, porém ela não é
mais usada na html5 pois não é semântica e essas alterações na forma
devem ser feitas com a css.

Para inserirmos um texto pequeno utilizamos a tag small.

Texto inserido

Um texto inserido é um texto que recebe ênfase pois foi inserido depois, para colocalo usamos a tag ins.

Texto deletado

Um texto deletado é um texto que deve ser lido mas não considerado.

Texto sobrescrito

É para inserir coisas do tipo a + b2

Texto subescrito

É para inserir coisas do tipo H2O
